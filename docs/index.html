Smart Anki Flashcards Documentation

Local-first, privacy-focused, AI-powered Chrome extension to create, learn, and export spaced-repetition flashcards. This page expands on installation, features, backups, shortcuts, and developer setup.

Overview

Turn any web page into spaced‑repetition flashcards with AI. Highlight text, press your shortcut, and save well‑formed Q/A cards into decks. Review with a keyboard‑first flow and track progress with a gamified dashboard.

Easy learning

Keyboard‑first review with spaced repetition (Easy / Good / Again). Learn faster with focused prompts and add images to reinforce concepts.

Easy card creation anytime

Select text on any web page and turn it into high‑quality Q/A cards in seconds.

Dashboard

Track sessions, streaks, and deck progress with a clean, motivating dashboard.

Sharing

Export single cards or entire decks to Anki for easy sharing or migration.

Customize for your learning

Tune settings, enable automatic backups, and set hotkeys to match your workflow.

Install & Use (No code)

  1. Get the files: unzip a prebuilt dist/ or build yourself (see Build & Ship).
  2. Open Chrome → chrome://extensions → enable Developer mode → Load unpacked → select the dist/ folder.
  3. Pin the extension from the toolbar Extensions menu.
  4. Assign shortcuts at chrome://extensions/shortcuts (defaults below).
  5. Optional: For silent backups, in chrome://settings/downloads turn OFF “Ask where to save each file before downloading”.
  6. Use it: highlight text → press Create Cards shortcut → choose deck → review and save.

Defaults

Create: Mac Command+Shift+Q, Windows/Linux Alt+Shift+Q. Reload: Command/Alt+Shift+R. Close: Command/Alt+Shift+X.

Customize in chrome://extensions/shortcuts.

Technical details

Automatic backups: Enable scheduled backups and use “Backup now” from the Options or Side Panel Settings.

Privacy-first: Data never leaves your browser. Database lives in IndexedDB via sql.js (WASM).

Tech stack: React, TypeScript, Tailwind, Vite, sql.js.

Project Structure

  • Side panel UI: src/scripts/sidepanel/sidepanel.html, src/scripts/sidepanel/index.tsx
  • Options UI: src/scripts/options/options.html, src/scripts/options/index.tsx
  • Service worker: src/scripts/service-worker/service-worker.ts
  • Content script: src/scripts/content/index.tsx
  • SQLite service: src/utils/sqlite.ts
  • Browser utilities: src/utils/browser.ts
  • Manifest generator: src/manifest.ts

Build & Ship

  1. Run: npm run build
  2. Verify dist/ and test in Chrome via Load unpacked
  3. Create a zip of dist for the Chrome Web Store

Vite builds entries for Options, Side Panel, and Service Worker. Post-build, the manifest is generated from project metadata, and sql.js WASM is copied into the bundle.

Security, Privacy, and CSP

Only the text you select is sent to AI providers. Everything else stays on your device. Permissions include downloads, alarms, notifications, storage, scripting, tabs, activeTab, sidePanel, commands, contextMenus, and host permissions for http/https to read selections.

CSP allows connect-src to AI endpoints, data/blob images, and wasm execution for sql.js.

FAQ

How do I use the AI feature for free?

You can use Google’s Gemini models with a generous free tier via Google AI Studio. You don't even need to input credit card.

  1. Go to Google AI Studio and sign in.
  2. Create an API key at API Keys.
  3. Copy your key and open the extension’s Settings → AI Provider → choose Gemini → paste your key.
  4. Save. Now card generation will use Gemini.

Free tier details: see Gemini pricing. The free tier is generous for personal use but subject to change.

How do I level up?
How you earn XP
  • Again = 0 XP
  • Good = +10 XP
  • Easy = +20 XP
  • XP adds up every time you rate a card after seeing the answer.
How levels work
  • Every 200 XP = +1 level.
  • Formula: Level = 1 + floor(total XP ÷ 200).
  • Example: 230 XP → Level 2 with 30/200 toward Level 3.
Progress bar to next level
  • The purple bar shows how far you are through the current 200‑XP step.
  • “+XP today” and “+XP last 7d” show recent gains.
Streaks (separate from levels)
  • Streak = number of days in a row with at least one review.
  • Streaks don’t change XP or levels; they’re just a motivation metric.
Notes
  • No daily caps and no penalties: you simply gain XP from Good/Easy ratings.
  • Keep studying: more Good/Easy ratings = faster level ups.
Does it work on Mac?
Yes. Default shortcuts on Mac use Command+Shift (e.g., Command+Shift+Q to create from selection).
IMPORTANT! PLEASE READ! Where are my cards stored?
Cards are stored locally using a lightweight SQL database running in the browser (WebAssembly). No server required. However, if you clear your browser, data will be lost. Please ensure you back up your data by downloading it to your local machine regularly.
Will you read my browsing data?
The extension processes only the text you explicitly select to create cards. Everything else stays on your device.
Can I change shortcuts?
Yes. Open chrome://extensions/shortcuts to customize keys for your system.
